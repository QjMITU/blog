java web三大组件

1、之一的监听器，Listener

servlet四大域

ServletContext ：这个域的范围是整个web应用

Request： 这个域的范围是一个请求过程中

Session：这个域的范围是 一次会话

PageContext： 这个域的范围是对应jsp页面。

  • 我们要创建监听器

    • javaweb提供的监听器接口有：

      • ServletContextListener

        用来监听ServletContext，对web项目启动时进行一些初始化操作，以及当web项目关闭时，进行一些收尾工作。

      • ServletContextAttributeListener

        ​ 该监听器监听ServletContext中数据的变化

      • ServletRequestListener

        用来监听ServletRequest对象的创建和销毁

      • ServletRequestAttributeListener

        监听ServletRequest域中的数据

    创建自己的监听，我们只需要实现以上的接口，然后实现对应的方法即可。

  • 配置web.xml

  • 下去任务，自己了解一下HttpSessionListener等其他监听器

cookie保存数据是通过 key-value的形式
